# gradeCalculation
Kullanıcıdan Vize ve Final notlarını isteyelim. Vize ve Final notlarının ortalaması 45'den küçük ise kullanıcıdan bütünleme notu
alalım ve tekrar 3 notun ortalamasına bakalım. Ortalama 45'den büyük ise geçti, 45 küçük ise kaldı, eğer 2 notun ortalaması 45'den
büyük ise geçti cevaplarını veren program.
